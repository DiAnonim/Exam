Игра "Пятнашки"
    Это реализация классической игры "Пятнашки" на Python с использованием Tkinter 
    для графического интерфейса. Игра заключается в перестановке квадратных плиток 
    с числами в сетке 4x4, перемещая их в пустое место.


Как играть:
    Для начала игры запустите файл main.py. 
    

Запуск приложения:
    Для запуска приложения выполните следующие шаги:
    1. Убедитесь, что у вас установлен Python (рекомендуется версия 3.x).
    2. Убедитесь, что у вас установлены необходимые библиотеки: Tkinter, requests, aiohttp, BeautifulSoup.
    3. Запустите файл main.py. Вам будет предложено меню, где вы сможете начать новую игру, 
        продолжить сохраненную, получить информацию о правилах игры или выйти из игры.


Инструкции по игре:
    1. Главное меню:
        "Start Game": Начать новую игру.
        "Continue": Продолжить игру.
        "Info": Получить информацию о правилах игры.
        "Exit": Выйти из приложения.

    2. Игровой процесс:
        Цель игры - упорядочить числа от 1 до 15 по порядку, оставив пустую ячейку в правом нижнем углу.
        Для перемещения чисел используйте пустую ячейку, нажимая на числа рядом с ней.
        Игра заканчивается, когда все числа стоят в правильном порядке.
        
    3. Победа:
        Когда все числа стоят в правильном порядке, игрок выигрывает.



Структура проекта:

    1. main.py: 
        Основной файл для запуска приложения. Создает экземпляр 
        класса MenuWindow и запускает главный цикл Tkinter.

    2. MenuWindowClass.py:
        Содержит класс MenuWindow, который представляет окно главного меню игры. 
        Позволяет пользователю начать новую игру, продолжить игру, 
        получить информацию о правилах игры и выйти из игры.

    3. StartGameClass.py: 
        Содержит класс StartGame, который обрабатывает логику игры "Пятнашки".
        Он позволяет игрокам перемещать плитки, подсчитывать ходы, 
        проверять условие победы и сохранять/загружать прогресс игры.

    4. InfoWebScraperClass.py:
        Содержит класс InfoWebScraper, который используется
        для извлечения информации о правилах игры с веб-сайта.

    5. WorkFileClass.py:
        Содержит класс WorkWithFile, который 
        отвечает за сохранение и загрузку данных игры из файла.

    6. ConfigureWindowClass.py:
        Содержит класс ConfigureWindow, который предоставляет функциональность 
        для настройки внешнего вида и поведения окон Tkinter, используемых в игре.
